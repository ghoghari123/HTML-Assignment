CSS Combinators Examples


[ 1 ] ( ~ ) The ~ (general sibling) selector applies CSS to all "span" elements that come after a "div" element at the same level.

[ 2 ] ( + ) The + (adjacent sibling) selector applies CSS to the first "p" element that immediately comes after a "div" element at the same level.


This paragraph is inside div (Not affected).

This paragraph comes immediately after div (Affected by + selector).
